Guia de Configuração: Google Planilhas & Apps Script

Passo 1: Preparar a Planilha (No Computador)

  1. Abra o Google Planilhas e crie uma planilha em branco.
  2. Monte a estrutura simples:
    • Coluna A: Suas tarefas diárias (ex: "Fio dental", "Estudar", "Treinar").
    • Coluna B: Selecione as células vazias, vá em Inserir > Caixa de seleção.
    • Coluna C: Digite os pontos de cada tarefa (ex: 10, 20, 15).
    • Célula D2: Digite o número 0 (onde ficará sua pontuação semanal).

Passo 2: Instalar os Códigos (No Apps Script)

  1. No menu superior da planilha, clique em Extensões > Apps Script.
  2. Apague qualquer código que estiver lá e cole este script completo:
Código.gs 
// 1. FUNÇÃO: Soma os pontos das tarefas concluídas usando a Coluna C e limpa as caixas de seleção
function resetDiarioETotalizador() {
  var planilha = SpreadsheetApp.getActiveSpreadsheet();
  var aba = planilha.getActiveSheet();
  
  var ultimaLinha = aba.getLastRow();
  if (ultimaLinha < 2) return; // Sem tarefas cadastradas
  
  var checkboxes = aba.getRange("B2:B" + ultimaLinha).getValues();
  var pontos = aba.getRange("C2:C" + ultimaLinha).getValues();
  var pontosDoDia = 0;
  
  for (var i = 0; i < checkboxes.length; i++) {
    if (checkboxes[i][0] === true) {
      var valorPonto = parseFloat(pontos[i][0]) || 0;
      pontosDoDia += valorPonto;
    }
  }
  
  var celulaPontosSemanal = aba.getRange("D2");
  var pontosAtuais = celulaPontosSemanal.getValue() || 0;
  
  celulaPontosSemanal.setValue(pontosAtuais + pontosDoDia);
  aba.getRange("B2:B" + ultimaLinha).setValue(false); // Desmarca tudo para o dia seguinte
}

// 2. FUNÇÃO: Zera a pontuação semanal para começar a nova semana
function resetPontuacaoSemanal() {
  var planilha = SpreadsheetApp.getActiveSpreadsheet();
  var aba = planilha.getActiveSheet();
  aba.getRange("D2").setValue(0); // Zera o placar em D2
}

Passo 3: Configurar as Automações (Acionadores)

No painel do Apps Script, clique no ícone de Relógio (Acionadores) no menu esquerdo e crie dois acionadores:

⏰ Acionador 1: Reset Diário (Soma pontos e limpa caixas)

  • Clique em Adicionar acionador (canto inferior direito).
  • Função que deve ser executada: resetDiarioETotalizador
  • Origem do evento: Baseado no tempo
  • Tipo de acionador: Temporizador diário
  • Hora do dia: Escolha um horário da madrugada (ex: 1h às 2h)
  • Clique em Salvar e autorize as permissões da conta.

⏰ Acionador 2: Reset Semanal (Zera os pontos no domingo)

  • Clique em Adicionar acionador novamente.
  • Função que deve ser executada: resetPontuacaoSemanal
  • Origem do evento: Baseado no tempo
  • Tipo de acionador: Temporizador semanal
  • Dia da semana: Domingo
  • Hora do dia: Escolha Meia-noite à 1h
  • Clique em Salvar.

Passo 4: Como usar no Celular?

Como tudo roda na nuvem do Google Drive, basta baixar o aplicativo gratuito do Google Planilhas no seu smartphone, logar com a mesma conta e pronto. Você preenche as checkboxes pelo celular e o script rodará automaticamente à noite.
